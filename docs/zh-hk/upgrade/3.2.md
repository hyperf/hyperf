# 3.2 升級指南

- 3.2 版本主要修改了 `PHP` 最低版本為 `8.2`。
- 升級 `symfony/*` 組件至支持 `^6.0 || ^7.0`。
- 升級 `phpunit/phpunit` 至 `^11.0`。

## 不兼容變更

1. `async-queue` 組件將 `JobInterface::getQueueName()` 重命名為 `getPoolName()`。

```php
<?php
// 變更前
class CustomJob extends \Hyperf\AsyncQueue\Job
{
    public function getQueueName(): string
    {
        return 'custom';
    }
}

// 變更後
class CustomJob extends \Hyperf\AsyncQueue\Job
{
    public function getPoolName(): string
    {
        return 'custom';
    }
}
```

2. `logger` 配置結構已變更。請參考 [#7563](https://github.com/hyperf/hyperf/pull/7563)。

3. `cache` 配置結構已變更。請參考 [#7594](https://github.com/hyperf/hyperf/pull/7594)。

4. 使用 `swow` 時，需要新增 response emitter 依賴。

```php
<?php
// config/autoload/dependencies.php

use Hyperf\Contract\ResponseEmitterInterface;
use Hyperf\Engine\ResponseEmitter;

return [
    ResponseEmitterInterface::class => ResponseEmitter::class,
];
```

## 依賴升級

- 升級 php 版本至 `>=8.2`。
- 升級 `elasticsearch/elasticsearch` 至 `^8.0 || ^9.0`。
- 升級 `nikic/php-parser` 至 `5.6`。
- 升級 `symfony/*` 組件至 `^6.0 || ^7.0`。
- 升級 `phpunit/phpunit` 至 `^11.0`。
- 升級 `google/protobuf` 至 `^3.6.1 || ^4.2`。
- 升級 `guzzlehttp/guzzle` 至 `^7.0`。

## 注意

如果你已經升級到 PHP 8.4，`fgetcsv` 和 `fputcsv` 方法現在都需要為 `escape` 參數提供默認值。

```php
fputcsv($fp, $fields, escape: '');
fgetcsv($fp, escape: '');
```
