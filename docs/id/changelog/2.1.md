## Dihapus

- Menghapus properti deprecated `$name` dari `Hyperf\Amqp\Builder`.

  * Melempar RunningInNonCoroutineException saat berjalan di luar konteks coroutine.

  * Menghapus method `execute`.
  * Method `initHeaders` akan mengembalikan `$headers`, alih-alih menetapkan "$headers" secara langsung ke klien.

- [#2909](https://github.com/hyperf/hyperf/pull/2909) Mengizinkan pengaturan middleware berulang.
