## Perubahan yang Tidak Kompatibel (BC breaks)

- Framework menghapus dukungan `@Annotation`, dan menggunakan annotation
  bawaan `PHP8` yaitu `Attribute`. Sebelum melakukan pembaruan, pastikan
  untuk memeriksa apakah proyek telah diganti dengan `Attribute`.

Script berikut dapat dijalankan untuk mengonversi `Doctrine Annotations` ke
`PHP8 Attributes`.

**Catatan: Script ini hanya dapat dieksekusi di bawah versi 2.2**

```shell
composer require hyperf/code-generator
php bin/hyperf.php code:generate -D app
```

- Script peningkatan Database Model

> Karena model base class telah menambahkan dukungan tipe untuk member variables,
> Anda perlu menggunakan script berikut untuk meningkatkannya ke versi baru.

```shell
composer require hyperf/code-generator
php vendor/bin/regenerate-models.php $PWD/app/Model
```

- Framework menambahkan lebih banyak batasan tipe ke class library, sehingga
  saat memperbarui dari `2.2` ke `3.0`, Anda perlu menjalankan pemeriksaan
  statis untuk memastikan semuanya berfungsi.

```shell
composer analysis
```

- Framework mengubah `Http status` yang dikembalikan oleh `gRPC Server` sesuai
  dengan spesifikasi `gRPC`. Status ditetapkan pada 200, dan `gRPC Server`
  mengembalikan `status code` yang sesuai. Peningkatan layanan ke versi 3.x
