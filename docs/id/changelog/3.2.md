# v3.2.0 - 2026-06-07

## Perubahan Breaking

1. Perubahan BC pada komponen Carbon
    - `Carbon::createFromTimestamp()` tidak lagi membaca zona waktu default
      melalui `date_default_timezone_get()` untuk `v3.0`.
    - `Carbon::diffInDays()` tidak lagi mengembalikan nilai absolut secara
      default. Gunakan parameter `$absolute` untuk mengontrol apakah hasilnya
      absolut.

2. [#7625](https://github.com/hyperf/hyperf/pull/7625) Mengubah nama
   `JobInterface::getQueueName()` menjadi `getPoolName()` pada komponen
   async-queue untuk konsistensi terminologi.

```php
<?php
// Sebelum
class CustomJob extends \Hyperf\AsyncQueue\Job
{
    public function getQueueName(): string
    {
        return 'custom';
    }
}

// Sesudah
class CustomJob extends \Hyperf\AsyncQueue\Job
{
    public function getPoolName(): string
    {
        return 'custom';
    }
}
```

```php
<?php

use Carbon\Carbon;

$t = time();

# Penggunaan yang break
Carbon::createFromTimestamp($t, date_default_timezone_get());

# Penggunaan yang benar
Carbon::createFromTimestamp($t, date_default_timezone_get());
```

3. Struktur konfigurasi `logger` telah diubah. Silakan merujuk ke
   [#7563](https://github.com/hyperf/hyperf/pull/7563).

```php
<?php

// Sebelum
return [
    'default' => [
        'driver' => 'daily',
        'path' => BASE_PATH . '/runtime/logs/hyperf.log',
        'level' => env('LOG_LEVEL', 'debug'),
        'days' => 7,
    ],
];

// Sesudah
return [
    'default' => 'default',
    'channels' => [
        'default' => [
            'driver' => 'daily',
            'path' => BASE_PATH . '/runtime/logs/hyperf.log',
            'level' => env('LOG_LEVEL', 'debug'),
            'days' => 7,
        ],
        // Tambahkan channel kustom Anda di sini
    ],
];
```

4. Struktur konfigurasi `cache` telah diubah. Silakan merujuk ke
   [#7594](https://github.com/hyperf/hyperf/pull/7594).

```php
<?php
// Sebelum
return [
    'default' => [
        'driver' => RedisDriver::class,
        'packer' => PhpSerializerPacker::class,
        'prefix' => 'c:',
    ],
];

// Sesudah
return [
    'default' => env('CACHE_DRIVER', 'default'),
    'stores' => [
        'default' => [
            'driver' => RedisDriver::class,
            'packer' => PhpSerializerPacker::class,
            'prefix' => 'c:',
        ],
    ],
];
```

5. Menambahkan dependensi saat menggunakan `swow`.

`config/autoload/dependencies.php`

```php
<?php
use Hyperf\Contract\ResponseEmitterInterface;
use Hyperf\Engine\ResponseEmitter;

return [
    ResponseEmitterInterface::class => ResponseEmitter::class,
];
```

## Peringatan

1. Jika Anda telah meng-upgrade ke PHP 8.4, harap perhatikan bahwa baik metode
   `fgetcsv` maupun `fputcsv` sekarang mewajibkan penyediaan nilai default untuk
   parameter `escape`.

```
fputcsv($fp, $fields) harus diubah menjadi fputcsv($fp, $fields, escape: '')
fgetcsv($fp) harus diubah menjadi fgetcsv($fp, escape: '')
```
