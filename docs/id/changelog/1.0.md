# Changelog

# v1.0.16 - 2019-09-20

## Ditambahkan

- [#565](https://github.com/hyperf/hyperf/pull/565) Menambahkan konfigurasi opsi
  (options config) untuk redis.
- [#580](https://github.com/hyperf/hyperf/pull/580) Menambahkan fitur kontrol
  konkurensi coroutine.

## Diperbaiki

- [#564](https://github.com/hyperf/hyperf/pull/564) Memperbaiki kesalahan
  typehint ketika `Coroutine\Http2\Client->send` gagal.
- [#567](https://github.com/hyperf/hyperf/pull/567) Memperbaiki kegagalan
  `getReturnType` pada rpc-client ketika namanya tidak sama dengan service.
- [#571](https://github.com/hyperf/hyperf/pull/571) Memperbaiki request
  berikutnya yang terpengaruh setelah menggunakan stopPropagation.
- [#579](https://github.com/hyperf/hyperf/pull/579) Inisialisasi dinamis
  metadata snowflake, memperbaiki masalah saat menggunakan snowflake dalam mode
  command (misalnya `di:init-proxy`) yang akan menghubungkan ke server redis
  dan menunggu timeout.

## Diubah

- [#583](https://github.com/hyperf/hyperf/pull/583) Melempar
  `GrpcClientException` ketika `BaseClient::start` gagal.
- [#585](https://github.com/hyperf/hyperf/pull/585) Melempar exception ketika
  eksekusi gagal di task worker.
